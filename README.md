# Digital Study Library (React)

A premium academic study platform built with React, Tailwind CSS, and Framer Motion.

## Features
- **Library Dashboard**: A modern grid view of available study materials.
- **Interactive Reader**: Smooth page-by-page reading experience.
- **Restricted Access**: Built-in logic to limit free reading to 3 pages.
- **Contact Integration**: Professional modal for lead generation and full access requests.
- **Responsive Design**: Optimized for all screen sizes with a clean "Academic" aesthetic.

## Tech Stack
- **Framework**: Vite + React
- **Styling**: Tailwind CSS
- **Icons**: Lucide-React
- **Animations**: Framer Motion

## Getting Started

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Run Development Server**
   ```bash
   npm run dev
   ```

3. **Build for Production**
   ```bash
   npm run build
   ```

## Design System
- **Primary Color**: Academic Blue (`#2563eb`)
- **Backgrounds**: Slate 50/100 for depth, White for content.
- **Typography**: Inter (Sans-serif) for maximum readability.
